## Asset Categories

### UTXO-based Cryptocurrencies
    
UTXO (Unspent Transaction Output) blockchains like Bitcoin track ownership through transaction outputs.

- **[Bitcoin (BTC)](/assets/bitcoin)** - The original cryptocurrency
- **[Litecoin (LTC)](/assets/litecoin)** - Faster block times and different hashing algorithm
- **[Bitcoin Cash (BCH)](/assets/bitcoin-cash)** - Increased block size for more transactions
- **[Dogecoin (DOGE)](/assets/dogecoin)** - Meme-coin with a passionate community

### Account-based Cryptocurrencies
    
Account-based blockchains like Ethereum track balances through accounts rather than UTXOs.

- **[Ethereum (ETH)](/assets/ethereum)** - Smart contract platform
- **[Ripple (XRP)](/assets/ripple)** - Fast settlement network

### Tendermint-based Cryptocurrencies
    
Tendermint blockchains are part of the Cosmos ecosystem using the Tendermint consensus engine.

- **[Cosmos (ATOM)](/assets/cosmos)** - Internet of blockchains
- **[Osmosis (OSMO)](/assets/osmosis)** - Automated market maker (AMM) protocol

## SDK Integration

The KeepKey SDK provides a consistent interface for all supported cryptocurrencies:

```tsx filename="example-integration.tsx" {5-10}
import { KeepKeySdk } from '@keepkey/keepkey-sdk';

// Initialize SDK
const sdk = await KeepKeySdk.create({
  apiKey: "your-app-name",
  pairingInfo: {
    name: "Your Application",
    imageUrl: "https://example.com/logo.png",
    url: window.location.origin,
  }
});

// Get Bitcoin address
const btcAddress = await sdk.address.bitcoin.get();

// Send transaction
const txid = await sdk.transaction.bitcoin.send({
  to: "1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa",
  amount: "0.001"
});
```
