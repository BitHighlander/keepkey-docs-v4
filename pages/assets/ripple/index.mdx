# Ripple (XRP)

## Overview
Ripple (XRP) is a digital payment protocol and cryptocurrency designed to facilitate cross-border money transfers with minimal fees and fast transaction times. KeepKey provides secure storage and transaction capabilities for XRP, allowing users to manage their Ripple with hardware-level security.

## Specifications

| Property | Value |
|----------|-------|
| Symbol | XRP |
| Decimal Places | 6 |
| Network Type | Account-based |
| Default Derivation Path | m/44'/144'/0'/0/0 |
| Address Format | 25-35 character string beginning with "r" |

## Derivation Paths

Ripple uses the following derivation path:

| Purpose | Path Format | Example |
|---------|-------------|---------|
| Standard (BIP44) | m/44'/144'/0'/0/0 | [0x8000002c, 0x80000090, 0x80000000, 0, 0] |

## XRP Considerations

- XRP addresses require a minimum balance (reserve) of 10 XRP to be activated
- Transactions typically settle in 4-5 seconds
- XRP has a fixed supply with no mining or inflation
