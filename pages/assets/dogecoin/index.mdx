# Dogecoin

## Overview
Dogecoin (DOGE) is a cryptocurrency that started as a meme but has grown into a significant digital asset with a strong community. KeepKey provides secure storage and transaction capabilities for Dogecoin, allowing users to safely manage their DOGE holdings.

## Specifications

| Property | Value |
|----------|-------|
| Symbol | DOGE |
| Decimal Places | 8 |
| Network Type | UTXO |
| Default Derivation Path | m/44'/3'/0'/0/0 |
| Address Format | Legacy (P2PKH) |

## Derivation Paths

Dogecoin uses the following derivation path:

| Purpose | Path Format | Example |
|---------|-------------|---------|
| Standard (BIP44) | m/44'/3'/0'/0/0 | [0x8000002c, 0x80000003, 0x80000000, 0, 0] |
