# Bitcoin Cash

## Overview
Bitcoin Cash (BCH) is a cryptocurrency that was created as a result of a hard fork from Bitcoin in 2017. It aims to provide faster transaction times and lower fees compared to Bitcoin. KeepKey supports secure storage and transactions for Bitcoin Cash.

## Specifications

| Property | Value |
|----------|-------|
| Symbol | BCH |
| Decimal Places | 8 |
| Network Type | UTXO |
| Default Derivation Path | m/44'/145'/0'/0/0 |
| Address Format | Cash Address format (prefix: bitcoincash:) |

## Derivation Paths

Bitcoin Cash uses the following derivation path:

| Purpose | Path Format | Example |
|---------|-------------|---------|
| Standard (BIP44) | m/44'/145'/0'/0/0 | [0x8000002c, 0x80000091, 0x80000000, 0, 0] |
