# Osmosis

## Overview
Osmosis (OSMO) is a decentralized, cross-chain AMM (Automated Market Maker) protocol built on the Cosmos SDK. It allows users to create liquidity pools and exchange assets within the Cosmos ecosystem. KeepKey provides secure storage and transaction capabilities for Osmosis.

## Specifications

| Property | Value |
|----------|-------|
| Symbol | OSMO |
| Decimal Places | 6 |
| Network Type | Tendermint |
| Default Derivation Path | m/44'/118'/0'/0/0 |
| Address Format | bech32 (prefix: osmo) |

## Derivation Paths

Osmosis uses the following derivation path:

| Purpose | Path Format | Example |
|---------|-------------|---------|
| Standard (BIP44) | m/44'/118'/0'/0/0 | [0x8000002c, 0x80000076, 0x80000000, 0, 0] |
