# Cosmos

## Overview
Cosmos (ATOM) is a cryptocurrency that powers an ecosystem of blockchains designed to scale and interoperate with each other. KeepKey provides secure storage and transaction capabilities for Cosmos, allowing users to manage their ATOM with hardware-level security.

## Specifications

| Property | Value |
|----------|-------|
| Symbol | ATOM |
| Decimal Places | 6 |
| Network Type | Tendermint |
| Default Derivation Path | m/44'/118'/0'/0/0 |
| Address Format | bech32 (prefix: cosmos) |

## Derivation Paths

Cosmos uses the following derivation path:

| Purpose | Path Format | Example |
|---------|-------------|---------|
| Standard (BIP44) | m/44'/118'/0'/0/0 | [0x8000002c, 0x80000076, 0x80000000, 0, 0] |
