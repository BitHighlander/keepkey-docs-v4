## Setup Process

### 1. Software Installation

<div style={{ display: 'flex', justifyContent: 'center', marginBottom: '2rem' }}>
    <Image
        src="/Art/ElectrumImage.jpeg"
        alt="KeepKey Software Setup"
        width={400}
        height={300}

</div>

1. **Download Software**
   - Get the latest KeepKey Desktop from [keepkey.com/get-started](https://www.keepkey.com/get-started)
   - Available for Windows, macOS, and Linux
   - Verify digital signatures (recommended)

2. **System Requirements**
   - Windows 10 or later
   - macOS 10.13 or later
   - Linux (major distributions)
   - USB 2.0+ port

### 2. Device Verification

1. **Security Check**
   - Inspect security seal integrity
   - Contact support if seal is broken
   - Verify package contents

   ![Security Seal](/boxseal.png)

2. **Package Contents**
   - KeepKey hardware wallet
   - USB-C cable
   - Recovery phrase card
   - Warranty information

   ![Package Contents](/box.jpg)

### 3. Initial Setup

1. **Physical Connection**
   - Use provided USB cable
   - Connect to computer directly
   - Avoid USB hubs if possible

   ![Device Connection](/plugin.jpg)

2. **Software Setup**
   - Launch KeepKey Desktop
   - Follow on-screen prompts
   - Complete firmware update if required

> **Critical Security Note:** Your recovery phrase will be displayed ONCE during setup. Write it down immediately and store it securely.

### 4. Device Configuration

1. **Firmware Update**
   - Allow initial update
   - Follow both device and app prompts
   - Don't disconnect during update

2. **Windows-Specific Notes**
   - Click "More info" if prompted
   - Select "Run anyway" for installation
   - Follow security prompts

### 5. Recovery Phrase Backup

<div style={{ display: 'flex', justifyContent: 'center', marginBottom: '2rem' }}>
    <Image
        src="/Art/RecoveryImage.jpeg"
        alt="Recovery Phrase Backup"
        width={400}
        height={300}
    />
</div>

1. **Documentation**
   - Use provided backup card
   - Write clearly in permanent ink
   - Verify each word carefully
   - Store in secure location

2. **Security Tips**
   - Never store digitally
   - Consider metal backup solutions
   - Keep multiple secure copies
   - Test recovery process
